VIM分屏功能

Posted on 2011-09-27 In 软件

分屏启动Vim

使用大写的O参数来垂直分屏。
vim -On file1 file2 …
使用小写的o参数来水平分屏。
vim -on file1 file2 …
注释: n是数字，表示分成几个屏。

关闭分屏

关闭当前窗口。
Ctrl+W c
关闭当前窗口，如果只剩最后一个了，则退出Vim。
Ctrl+W q
