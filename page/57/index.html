装修——强弱电同盒（转）

Posted on 2010-07-17 In 装修

强弱电的间距一直是电改的常见问题，本图片更为突出，该处的强弱电线路同进一个线盒内，问题就更加严重。强弱电线路在一起由于干涉会影响今后的使用，同时我们还要看规范与实际情况。规范：《住宅装饰装修工程施工规范》（GB50327-2001） 16.3.5、电源线与通讯线不得穿入同一根管内。16.3.6、电源线及插座与电视线及插座的水平间距不应小于500mm。实际在家装中，这个距离很难得到保证，通过实际现在通行强弱电的间距不得小于300mm，个人觉得大于200mm即可，当然，采用弱电金属管可以将间距缩短（金属管有屏蔽作用）但决不能在同一管道和线盒内。

fedora ssh 访问

Posted on 2010-07-15 In fedora

为fedora 安装 ssh
yum install openssh-server

启动ssh
/etc/init.d/sshd start

ps:如果启用了防火墙，请打开ssh 22端口，遇到一个很奇怪的问题，设置防火墙后似乎没有生效，在关闭防火墙再次打开时，应用设置才生效，叵测！
